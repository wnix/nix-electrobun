Welcome to your Electrobun app! This framework combines Bun's lightning-fast runtime with native desktop capabilities.

What is Electrobun?

  • Fast: Built on Bun's lightning-fast JavaScript runtime
  • Native: Access to system APIs like menus, trays, and file dialogs
  • Cross-platform: Works on macOS, Windows, and Linux
  • Web-based UI: Use familiar HTML, CSS, and TypeScript for your interface
